# Support

- Usage questions and setup help: [GitHub Discussions](https://github.com/pycarrot/meetwise-local-sqlite/discussions)
- Reproducible bugs: [GitHub Issues](https://github.com/pycarrot/meetwise-local-sqlite/issues)
- Security reports: [Private vulnerability reporting](https://github.com/pycarrot/meetwise-local-sqlite/security/advisories/new)

Before asking for help, run:

```bash
node --version
ollama --version
ollama list
npm run check
```

Share version numbers and error messages only. Remove transcript text, speaker names, local paths, tokens, and other private information.
